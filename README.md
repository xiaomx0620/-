# 日语视频自动字幕生成工具

选一个日语视频，自动生成带中文翻译的SRT字幕文件，还能提前告诉它人名和术语怎么翻。

## 这个工具是干什么的？

你给它一个日语视频，它还你一个带时间轴的中文字幕文件（.srt）。

整个过程全自动：
1. 先用语音识别把视频里的日语对话转成文字
2. 然后调用多个翻译引擎把日语翻成中文
3. 最后生成一条一条带时间轴的字幕文件

## 它跟别的字幕工具比有什么不一样？

最大的特点：你可以提前告诉它人名和术语怎么翻。

比如你正在做《间谍过家家》的字幕，你可以在开始前告诉工具：

ロイド・フォージャー -> 劳埃德·福杰（黄昏）
ヨル・フォージャー -> 约尔·福杰（荆棘公主）
アーニャ -> 阿尼亚
西国 -> 西国
おはよう -> 早安

工具在翻译时会先在本地把这些词强制替换掉，再送去翻译引擎。

## 主要功能模块

阶段一：语音识别
- 默认使用 SenseVoiceSmall 云端模型（SiliconFlow 平台），支持日语
- 也可以切换到 Whisper 本地模型
- 支持主备切换

阶段二：本地术语替换（核心特色）
- 自动解析你填写的术语表（日文 -> 中文）
- 逐句匹配，长词优先替换
- 支持整句命中
- 使用占位符技术，防止替换内容被翻译引擎二次改写

阶段三：多引擎翻译 + 智能择优
- 同时调用多个翻译引擎：有道智云、百度翻译、DeepLX、SiliconFlow
- 每个引擎的结果会通过质量评分 + 相似度比较智能选择最优译文

阶段四：生成SRT字幕
- 自动生成标准格式的 .srt 字幕文件
- 保存在视频同目录下

## 安装与配置

环境要求：
- Windows 系统
- Python 3.9 或更高版本
- 网络连接

快速安装：
1. 下载本脚本，保存为 subtitle_tool.py
2. 打开终端，进入脚本所在目录
3. 运行 python subtitle_tool.py
4. 点击界面上的 部署环境 按钮，自动安装所有依赖包
5. 点击 设置，填写各翻译服务商的 API 密钥

如果自动安装失败，手动执行：
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple faster-whisper pydub av opencc-python-reimplemented requests tqdm

## 使用步骤

第一步：填写术语表
点击 选择视频 后，工具会弹出一个窗口让你填写术语表。

第二步：开始处理
点击确定后，工具自动完成语音识别、术语替换、翻译、生成字幕。

第三步：找到字幕文件
处理完成后，字幕文件保存在视频同一个文件夹里，文件名是 原视频名_字幕.srt。

## 常见问题

Q：为什么有些句子没翻译？
A：可能是翻译引擎返回了无效结果。如果所有引擎都失败，会保留日文原文。

Q：术语替换没生效？
A：检查术语表格式是否正确，必须是 日文 -> 中文。

Q：报错 Model does not exist？
A：检查 SiliconFlow 设置里的模型名称，推荐使用 FunAudioLLM/SenseVoiceSmall。

Q：处理速度很慢？
A：语音识别和翻译都需要联网调用 API，可以只启用一个翻译引擎来提速。
